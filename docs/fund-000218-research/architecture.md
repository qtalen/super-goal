### 1.2 分析方法

#### 定量分析（所有指标计算基于近 1 年/3 年/成立以来数据）

| 指标 | 公式/定义 | 用途 |
|------|-----------|------|
| 年化收益率 | `(期末净值/期初净值)^(1/年数) - 1` | 衡量长期回报 |
| 年化波动率 | 周收益率标准差 × √52 | 衡量风险水平 |
| 夏普比率 | `(年化收益率 - 无风险利率) / 年化波动率` | 风险调整后收益 |
| 最大回撤 | `Max(峰顶→谷底跌幅)` | 极端损失风险 |
| 信息比率 | `(基金年化收益 - 基准年化收益) / 跟踪误差` | 主动管理能力 |
| 跟踪误差 | 基金与基准日收益率差值的年化标准差 | 跟踪精准度 |
| Beta | 基金与金价的协方差 / 金价方差 | 对金价的敏感度 |
| 卡玛比率 | `年化收益率 / 最大回撤` | 收益/回撤性价比 |

#### 定性分析

| 维度 | 评估内容 | 信息来源 |
|------|----------|----------|
| 基金经理 | 从业年限、任职期表现、管理产品一致性 | 天天基金网、基金公告 |
| 管理团队 | 投研团队规模、公司整体实力、管理规模 | 公司官网、基金业协会 |
| 投资策略 | 跟踪策略透明度、费用合理性、规模适配度 | 基金合同、招募说明书 |
| 宏观环境 | 当前黄金定价逻辑（实际利率、美元、避险） | 美联储、世界黄金协会 |

### 1.3 评级框架

采用**多维度加权评分法**，最终得分映射到四个结论等级（详见第 4 节）。

## 4. 投资评级体系

### 4.1 评级维度与权重

| 维度 | 权重 | 评分内容 | 数据来源 |
|------|------|----------|----------|
| **A. 业绩表现** | 30% | 各阶段收益率水平、超额收益、同类排名百分位 | 需求2 |
| **B. 风险控制** | 25% | 最大回撤、波动率、夏普比率、卡玛比率 | 需求2 |
| **C. 跟踪效率** | 15% | 跟踪误差、费率水平、规模适配度 | 需求1、3 |
| **D. 管理团队** | 10% | 基金经理经验、任职期表现、公司整体实力 | 需求4 |
| **E. 宏观适配度** | 20% | 当前宏观环境对黄金的利好/利空程度 | 需求5 |

**合计：100%**

### 4.2 评分标准（百分制，每维度独立评分）

#### A. 业绩表现（30 分制，先评 0-100 再乘以权重）

| 评分区间 | 标准 |
|----------|------|
| 90-100 | 近1/3年收益率均位居同类前10%，大幅跑赢基准 |
| 75-89 | 位居同类前30%，跑赢基准 |
| 60-74 | 同类中游，与基准基本持平 |
| 40-59 | 同类后50%，阶段性跑输基准 |
| <40 | 持续跑输基准，同类排名后10% |

#### B. 风险控制（25 分制，先评 0-100 再乘以权重）

| 评分区间 | 标准 |
|----------|------|
| 90-100 | 最大回撤显著小于同类，夏普比率>1，波动率低于平均 |
| 75-89 | 风险指标优于同类平均，夏普比率0.5-1 |
| 60-74 | 风险指标与同类基本一致 |
| 40-59 | 风险指标劣于同类平均 |
| <40 | 最大回撤过大，夏普比率为负或极低 |

#### C. 跟踪效率（15 分制，先评 0-100 再乘以权重）

| 评分区间 | 标准 |
|----------|------|
| 90-100 | 年跟踪误差<0.5%，综合费率<0.5%，规模适中 |
| 75-89 | 年跟踪误差0.5%-1%，费率合理 |
| 60-74 | 年跟踪误差1%-2%，费率尚可 |
| 40-59 | 年跟踪误差>2%或费率偏高 |
| <40 | 跟踪误差过大或费率远高于同类 |

#### D. 管理团队（10 分制，先评 0-100 再乘以权重）

| 评分区间 | 标准 |
|----------|------|
| 90-100 | 基金经理从业>10年，任职期表现优异，公司规模前十 |
| 75-89 | 从业5-10年，任职期稳定，公司实力较强 |
| 60-74 | 从业3-5年，表现一般，中型公司 |
| 40-59 | 从业<3年，或高管频繁变动 |
| <40 | 基金经理频繁更换，或公司负面新闻多 |

#### E. 宏观适配度（20 分制，先评 0-100 再乘以权重）

| 评分区间 | 标准 |
|----------|------|
| 90-100 | 降息周期+高通胀+地缘风险高+美元弱势 → 强烈利好黄金 |
| 75-89 | 多数宏观因素利好黄金 |
| 60-74 | 宏观环境中性，多空因素均衡 |
| 40-59 | 多数宏观因素利空黄金（加息周期+强势美元） |
| <40 | 强烈利空黄金（快速加息+美元极度强势+风险偏好高） |

### 4.3 综合得分计算公式

```
总分 = A得分×30% + B得分×25% + C得分×15% + D得分×10% + E得分×20%
```

### 4.4 最终结论模板

| 总分区间 | 评级 | 结论模板 |
|----------|------|----------|
| ≥85 | **强烈推荐** | "国泰黄金ETF联接A综合表现优异，叠加当前宏观环境利好黄金，建议**超配**，配置比例可在总资产的15%-20%，适合作为避险资产配置。" |
| 70-84 | **推荐** | "该基金投资价值良好，建议**标配或适度超配**，配置比例可在总资产的10%-15%，适合中长期持有。" |
| 50-69 | **中性** | "基金表现中规中矩，宏观环境尚不明朗，建议**标配或观望**，配置比例不超过总资产的5%-10%。" |
| <50 | **回避** | "该基金或宏观环境存在明显不利因素，建议**低配或回避**，暂不建议买入。" |

### 5.3 数据汇集流程（需求1-5 → 需求6）

1. **需求2** 将需求1的基础信息与净值数据结合，产出业绩与风险指标
2. **需求3** 利用需求1的基金档案定位持仓数据，产出持仓分析
3. **需求4** 利用需求1的基金经理信息，扩展为管理团队评价
4. **需求5** 独立产出宏观环境研判
5. **需求6** 接收以上 4 路数据，执行评分公式：
   - 业绩表现分 ← 需求2（阶段收益、同类排名）
   - 风险控制分 ← 需求2（夏普、回撤、波动率）
   - 跟踪效率分 ← 需求1（费率）+ 需求3（跟踪误差）
   - 管理团队分 ← 需求4（经理履历、公司实力）
   - 宏观适配度分 ← 需求5（利率、通胀、美元、地缘）
6. 综合计算总分 → 映射为四档评级 → 生成结论模板

### 5.4 需求6 → 需求7（报告撰写流程）

1. 需求6的**综合评分卡**作为报告核心结论，填入第六章
2. 评分过程中的**关键论据**（各维度的强弱项）作为第六章"优势分析"和"风险提示"的内容
3. 需求7按第2节的**章节大纲**组织，将需求1-5的分析文本分别填入第一至五章
4. 第一章至第五章的每章末尾增加"小结"段落，提炼关键信息供第六章引用
5. 第六章的核心结论直接产出第七章
6. 附录A列出所有数据来源URL，附录B为标准免责声明，附录C附上第1.2节中的指标计算公式

### 5.5 质量控制要点

- 每个需求完成后需进行**数据交叉验证**（如：天天基金网的净值数据与晨星数据对比，确保无重大偏差）
- 定量指标计算使用统一的时间区间参数，避免不一致
- 定性评价需附明确依据，不得无据判断
- 评分过程需记录每个维度的得分理由，确保可追溯
- 最终报告需标明数据截止日期
