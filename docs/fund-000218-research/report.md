## 第一章  基金概况

### 1.1 基金基本信息档案

| 项目 | 内容 |
|------|------|
| **基金全称** | 国泰黄金交易型开放式证券投资基金联接基金 |
| **基金简称** | 国泰黄金ETF联接A |
| **基金代码** | 000218（前端） |
| **基金类型** | 指数型-其他（商品型-非QDII） |
| **风险等级** | 中高风险（R4） |
| **成立日期** | 2016年04月13日 |
| **基金管理人** | 国泰基金管理有限公司 |
| **基金托管人** | 中国工商银行股份有限公司 |
| **基金经理** | 艾小军（2016-04-13 至今，任职超10年） |
| **成立以来累计涨幅** | **+226.07%**（截至2026-07-07） |
| **最新净值** | 3.2607元（2026-07-07） |

该基金为国泰黄金ETF（518800）的**联接基金**，主要功能是为无法直接参与场内ETF交易的投资者提供场外投资渠道。投资者购买本基金，相当于间接持有上海黄金交易所Au99.99实物黄金。

### 1.2 基金规模与份额变化

基金规模呈现**爆发式增长**趋势：

| 时间节点 | 净资产规模 | 份额规模 | 同比变化 |
|----------|-----------|---------|---------|
| 2021年末 | ~1.32亿元 | 0.96亿份 | — |
| 2023年末 | ~7.20亿元 | 4.08亿份 | 黄金牛市启动 |
| 2024年末 | ~32.37亿元 | 14.45亿份 | 持续大规模净申购 |
| 2025年末 | ~120.03亿元 | 34.09亿份 | 爆发式增长 |
| **2026年一季度末** | **164.52亿元** | **44.61亿份** | 继续强劲增长 |

自2023年以来，伴随黄金价格持续上涨，基金规模从不到2亿元增长至超164亿元，**增长约80倍**。规模的快速扩张不仅未影响运作效率，反而通过摊薄固定成本进一步优化了跟踪表现。

### 1.3 费率结构分析

| 费用类型 | 费率 |
|----------|------|
| **管理费率** | 0.50%/年 |
| **托管费率** | 0.10%/年 |
| **销售服务费率** | 0.00%/年（A类无） |
| **综合运作费率** | **0.60%/年** |

申购费率（天天基金优惠后）：小于100万元仅**0.07%**，赎回费持有2年以上降至**0.00%**。综合费率在同类黄金ETF联接基金中处于**中等偏低**水平（同类范围0.60%-0.95%/年）。

### 1.4 跟踪标的说明

| 项目 | 内容 |
|------|------|
| **跟踪标的** | 黄金9999（上海黄金交易所Au99.99合约） |
| **业绩比较基准** | Au99.99合约收益率×95% + 银行活期存款税后收益率×5% |
| **年化跟踪误差** | **0.91%**（合同要求≤4%，同类平均2.13%） |
| **底层资产** | 穿透后100%对应上海黄金交易所认证的标准金Au99.99实物 |

本基金通过持有国泰黄金ETF（518800，上海证券交易所上市）实现对黄金资产的紧密跟踪。目标ETF底层资产为Au99.99现货实盘合约，**无杠杆、无衍生品**，风险透明度极高。

---

**第一章小结**：国泰黄金ETF联接A成立于2016年，是一只跟踪Au99.99黄金现货价格的被动联接基金。基金规模在黄金牛市中从约2亿元爆发增长至164亿元，综合费率0.60%/年在同类中偏低，跟踪精度行业领先（0.91%）。底层资产为实物黄金，透明度极高。

### 3.2 重仓持有明细

| 排名 | 持有标的 | 代码 | 类型 | 占比 | 说明 |
|:---:|---------|:---:|------|:---:|------|

前1大持仓占比93.59%，**集中度极高**。对于商品型基金，这是产品设计特征而非缺陷——投资者的目的是获取黄金价格敞口，集中持有正是实现这一目标的方式。

### 3.3 跟踪误差测算

| 指标 | 本基金 | 同类平均 | 合同要求 |
|------|:-----:|:-------:|:--------:|
| **年化跟踪误差** | **0.91%** | 2.13% | ≤4% |
| 日均跟踪偏离度 | 远低于0.35% | — | ≤0.35% |
| 2026Q1超额收益 | **+0.29%**（A类） | — | — |

**跟踪误差极低的六大原因**：
1. **底层资产简单**：单一Au99.99商品，无成分股调整等复杂因素
2. **目标ETF高度关联**：联接基金与ETF由同一管理人运作，成本最低化
3. **黄金租赁业务**：518800通过出租实物黄金获取租赁收入，部分抵消费用
4. **组合套利增强**：利用ETF一/二级市场价差套利，覆盖运作成本
5. **接近满仓运作**：93.59% ETF + 99.77% Au99.99，偏离仅来自现金和费用
6. **规模效应摊薄成本**：164亿元规模摊薄固定成本和交易佣金

| 评估维度 | 评价 |

| **投资目标达成** | ⭐⭐⭐⭐⭐ "紧密跟踪黄金资产，追求跟踪误差最小化"已充分实现 |

**第三章小结**：基金采用"联接+ETF+实物金"三层架构，93.59%持仓集中于国泰黄金ETF，穿透后100%对应Au99.99实物黄金。年化跟踪误差0.91%远优于同类（2.13%）和合同目标（4%），得益于底层资产简单、黄金租赁抵费用、组合套利增强和规模效应。策略执行效果**优秀**。

### 附录C：参考指标计算说明

| 指标 | 公式 | 说明 |
|------|------|------|
| **年化收益率** | (期末净值/期初净值)^(1/年数) - 1 | 衡量长期回报水平 |
| **年化波动率** | 周收益率标准差 × √52 | 衡量风险水平，数值越大波动越剧烈 |
| **夏普比率** | (年化收益率 - 无风险利率) / 年化波动率 | 衡量每单位风险获得的超额回报，>1为优秀 |
| **最大回撤** | Max(峰顶净值→谷底净值跌幅) | 衡量历史极端损失幅度 |
| **卡玛比率** | 年化收益率 / 最大回撤 | 衡量每单位最大回撤获得的年化回报 |
| **信息比率** | (基金年化收益 - 基准年化收益) / 跟踪误差 | 衡量主动管理能力，被动基金接近0即为良好 |
| **跟踪误差** | 基金与基准日收益率差值的年化标准差 | 衡量跟踪精准度，越低越好 |
| **Beta** | 基金与金价的协方差 / 金价方差 | 衡量对金价的敏感度，接近1.0为理想 |
| **无风险利率** | 中国1年期国债收益率（1.12%，2026-07-08） | 计算夏普比率的基准利率 |
