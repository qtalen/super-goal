| **研报参考** | Morgan Stanley / Goldman Sachs / Citi 等投行研报 (摘要) | 卖方观点交叉验证 |

### 1.2 分析方法

#### 定量分析
| 方法 | 说明 | 适用需求 |
|------|------|----------|
| **DCF 估值** | 基于最近12个月 (TTM) 自由现金流，分两阶段模型（5年高增长 + 永续增长 3%） | 需求 5（估值） |
| **PEG 比率** | 当前 P/E ÷ 未来 3-5 年预期 EPS CAGR | 需求 5（估值） |
| **P/S 比率** | 股价 / 每股营收，对比历史区间和同业 | 需求 5（估值） |
| **EV/EBITDA** | 企业价值 / EBITDA，排除折旧差异影响 | 需求 5（估值） |
| **FCF Yield** | 自由现金流 / 市值，对标无风险利率 | 需求 5（估值） |
| **收入增速分解** | 拆分为量价、产品组合、地理分布 | 需求 1, 2 |
| **盈利能力拆解** | 毛利率→运营利润率→净利润率逐层拆解，找出边际变化归因 | 需求 2 |
| **DuPont 分解** | ROE = 净利率 × 资产周转率 × 杠杆倍数 | 需求 2 |

#### 定性分析
| 方法 | 说明 | 适用需求 |
|------|------|----------|
| **波特五力模型** | 供应商议价力(台积电独供)、买方议价力(云厂商自研)、新进入者威胁(ASIC)、替代品威胁(AMD/推理专用芯片)、现有竞争强度 | 需求 4, 6 |
| **护城河评估** | 品牌(GPU/AI 标杆)、技术壁垒(CUDA 生态/3 百万开发者)、规模效应(研发投入领先同行)、转换成本(CUDA 锁定) | 需求 1, 4 |
| **SWOT 分析** | 综合内外部因素，为需求 7 提供评分输入 | 需求 7 |
| **管理层分析** | 黄仁勋战略执行力、持股比例、历史资本配置(回购/分红/并购) | 需求 1, 2 |

## 4. 投资评级体系

### 4.1 评级维度与权重

| 维度 | 权重 | 说明 |
|------|------|------|
| **财务质量** | 15% | 盈利能力、现金流健康度、资产负债状况、ROE 水平 |
| **成长性** | 25% | 营收/EPS 增速、市场空间、产品路线图动能（权重最高，因为是成长股） |
| **估值水平** | 20% | 当前估值相对于历史和同业的合理程度、与成长的匹配度 |
| **竞争壁垒** | 15% | CUDA 生态护城河、品牌效应、技术领先地位、客户转换成本 |
| **行业前景** | 15% | AI 芯片市场 TAM 增速、行业景气度、渗透率阶段 |
| **风险因素** | 10% | 地缘政治/竞争/客户集中/周期/估值泡沫风险综合评估 |

**总分 = Σ(维度得分 × 权重)**，满分 100 分。

### 4.2 各维度评分标准

#### 财务质量（15%）
| 分数 | 标准 |
|------|------|
| 5 (优秀) | 毛利率 >75%，净利率 >40%，ROE >50%，FCF 持续正增长，负债率 <20% |
| 4 (良好) | 毛利率 60-75%，净利率 30-40%，ROE 30-50% |
| 3 (一般) | 毛利率 45-60%，净利率 15-30%，ROE 15-30% |
| 2 (较差) | 毛利率 30-45%，净利率 5-15%，ROE 5-15% |
| 1 (差) | 毛利率 <30%，净利率 <5% 或亏损，FCF 为负 |

#### 成长性（25%）
| 分数 | 标准 |
|------|------|
| 5 (优秀) | 营收增速 >50%，3-5年预期 CAGR >30%，TAM 翻倍级增长 |
| 4 (良好) | 营收增速 25-50%，预期 CAGR 20-30% |
| 3 (一般) | 营收增速 10-25%，预期 CAGR 10-20% |
| 2 (较慢) | 营收增速 5-10%，预期 CAGR 5-10% |
| 1 (停滞) | 营收增速 <5% 或负增长 |

#### 估值水平（20%）
| 分数 | 标准 |
|------|------|
| 5 (低估) | P/E < 历史中位数 20% 以上，PEG < 1.0 |
| 4 (合理偏低) | P/E 接近历史中位数或略低，PEG 1.0-1.5 |
| 3 (合理) | P/E 处于历史中位数区间，PEG 1.5-2.0 |
| 4N (偏贵) | P/E > 历史中位数 20-50%，PEG 2.0-3.0 |
| 5N (高估) | P/E > 历史中位数 50%+，PEG > 3.0，FCF yield < 无风险利率 |

> 注：成长股估值中枢通常高于传统股，评分应结合行业特性进行判断。

#### 竞争壁垒（15%）
| 分数 | 标准 |
|------|------|
| 5 (极宽) | 近乎垄断地位 (>80% 份额)，CUDA 级生态锁定，3-5年无法追赶 |
| 4 (较宽) | 70-80% 份额，显著技术+生态领先，2-3年追赶窗口 |
| 3 (中等) | 50-70% 份额，存在竞品但领先优势明显 |
| 2 (较窄) | 30-50% 份额，竞争激烈 |
| 1 (无) | <30% 份额，无差异化优势 |

#### 行业前景（15%）
| 分数 | 标准 |
|------|------|
| 5 (朝阳) | AI 芯片市场高速扩张 (CAGR >30%)，渗透率 <30%，长期 >10 年增长周期 |
| 4 (景气) | CAGR 20-30%，渗透率 30-50% |
| 3 (平稳) | CAGR 10-20%，渗透率 50-70% |
| 2 (下行) | CAGR 5-10% 或增速趋缓 |
| 1 (夕阳) | CAGR <5% 或负增长，被替代风险 |

#### 风险因素（10%）— 分数越低风险越大
| 分数 | 标准 |
|------|------|
| 5 (低风险) | 各维度风险均较低，有明确对冲手段 |
| 4 (较低) | 存在 1-2 项中等风险，影响有限 |
| 3 (中等) | 存在 2-3 项中等-高风险，需持续关注 |
| 2 (较高) | 存在 1-2 项重大风险，可能实质性影响基本面 |
| 1 (高风险) | 存在不可控重大风险（如全面出口管制、技术替代） |

### 4.3 最终评级结论模板

| 总分区间 | 评级 | 含义 |
|----------|------|------|
| 85-100 | 强烈推荐买入 (Strong Buy) | 基本面优异+估值合理+成长确定性强，建议超配 |
| 70-84 | 买入 (Buy) | 基本面良好+估值合理，建议积极关注 |
| 55-69 | 持有 (Hold) | 基本面一般或估值偏高，建议保持现有仓位 |
| 40-54 | 减持 (Reduce) | 基本面走弱或估值显著偏高，建议降低仓位 |
| <40 | 卖出 (Sell) | 基本面恶化或严重高估，建议清仓 |

### 4.4 目标价格设定规则

- **基准情形**：基于 DCF 结果结合 Forward P/E × 一致预期 EPS 交叉验证
- **乐观情形**：基准情形 × 1.2（假设份额维持 + 超预期增长）
- **悲观情形**：基准情形 × 0.8（假设份额侵蚀 + 增长放缓）
- **最终目标价**：三种情形的概率加权平均（基准:乐观:悲观 = 50%:25%:25%）

## 7. 质量标准

### 7.1 数据引用规范
- 每个定量数据必须注明来源（如：`来源：NVDA FY2026 10-K, p.42`）
- 一致预期数据注明"截至 YYYY-MM-DD"的时间戳
- 图表和数据表使用 GFM 表格语法（兼容 Markdown 渲染）

### 7.2 分析深度要求
- 财务分析：需区隔一次性项目（如库存计提）与经常性项目，不应直接用 GAAP 数据做趋势分析
- 估值分析：至少两种独立方法交叉验证（DCF + 倍数法）
- 风险分析：每项风险需给出"可能性（高中低）× 影响程度（大小）"判断

### 7.3 报告风格
- 客观中立，避免夸张修辞
- 优缺点均需涉及，不得只讲利好
- 数据先行，结论在后，逻辑链条完整
- 每章结尾留一句过渡段，串联到下一章
