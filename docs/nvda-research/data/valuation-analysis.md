## 4. DCF估值模型（简化版）

### 4.1 模型设计

采用**两阶段模型**：
- **第一阶段（高增长期）**：前5年，各情景使用递减的增长率
- **第二阶段（永续增长期）**：第6年起，永续增长率

**关键共同假设**：

| 参数 | 数值 | 说明 |
|------|------|------|
| 流通股（稀释） | 24.51B | 基于FY2026年报 |
| 净现金 | $51.5B | 现金+短期投资$62.6B - 总债务$11.0B |
| 无风险利率 | 4.56% | 10年期美债收益率（2026-07-08） |
| Beta | 2.21 | 5年月度Beta（Yahoo Finance） |
| 权益风险溢价（ERP） | 5.0% | 美国大型股历史ERP |

**WACC计算**（CAPM模型）：
- CAPM成本权益 = 4.56% + 2.21 × 5.0% = **15.6%**
- 因净现金头寸（无净债务），WACC ≈ 成本权益
- 情景分析中使用9.5%-13%的WACC范围，反映市场实际使用的折现率（CAPM对高Beta股票的刻板估计往往偏高）

| 维度 | 判断 |
|------|------|

> 4. 使用FCF Yield溢价法：假设市场要求300bp的成长股溢价，则目标FCF Yield = 4.56% + 3.00% = 7.56%，对应市值为更低——这显然不适用于超级成长股

## 7. 估值综合判断

### 7.1 多维度评估

| 维度 | 信号 | 评价 |
|------|------|------|
| **相对历史估值** | **利好** | P/E低于5年中位数50%+，P/S低于5年中位数16%，均处于历史低位区域 |
| **相对成长性（PEG）** | **极大利好** | PEG 0.43-0.60，远低于1.0，在Mag7中最低 |
| **相对同业估值** | **中性偏利好** | Forward P/E与Mag7均值持平，但增速是其他公司的3-14倍 |
| **DCF绝对估值** | **预警** | 基准情景隐含价值$126（低于市价36%），但Bull Case $220覆盖当前价格 |
| **FCF Yield** | **中性** | 2.05-2.52%低于无风险利率，但对高成长股属正常范围，且处于历史高位 |
| **分析师情绪** | **利好** | ~94%买入以上评级，目标价均值$302（+54%），预期持续上修 |

### 7.2 核心矛盾：绝对估值 vs 相对估值

```
价格锚点（$197）
    │
    ├── DCF基准价值：$126（低估信号）
    │
    ├── Forward P/E × FY2027 EPS：$199（公允信号）
    │
    ├── Forward P/E × FY2028 EPS：$280（低估信号）
    │
    ├── PEG框架：PEG 0.48-0.60（极低估信号）
    │
    └── 分析师目标价均值：$302（低估信号）
```

**解释**：
1. DCF基准估值偏低，主要因为高Beta（2.21）导致WACC较高（10.5%+），而DCF对折现率极为敏感
2. DCF无法完全捕捉NVDA作为"AI基础设施垄断者"的战略溢价和增长期权价值
3. 跨期P/E对比显示，如果使用FY2028预期EPS，即使保持22x Forward P/E不变，一年后隐含股价为$280
4. **PEG框架是最有力的看多信号**——PEG 0.43-0.60在大型科技股中极为罕见

### 7.3 关键风险因素对估值的影响

| 风险因素 | 对估值的潜在影响 | 严重程度 |
|----------|----------------|---------|
| AI基础设施CapEx增速放缓 | P/E向历史均值回归（30x→20x） | ⚠️⚠️⚠️ |
| 竞争加剧（AMD/ASIC/自研芯片） | 市场份额→增长率→P/E三杀 | ⚠️⚠️⚠️ |
| 出口管制进一步升级 | 中国收入归零已反映，未反映全面脱钩 | ⚠️⚠️ |
| 产品周期过渡不顺 | 毛利率下降→EPS不达预期 | ⚠️⚠️ |
| FCF Yield持续低于无风险利率 | 高估值依赖低利率环境维持 | ⚠️ |

### 7.4 估值维度评分（参考architecture.md 第4节）

**评分标准回顾**：

| 分数 | 标准 |
|------|------|
| 5 (低估) | P/E < 历史中位数20%以上，PEG < 1.0 |
| 4 (合理偏低) | P/E接近历史中位数或略低，PEG 1.0-1.5 |
| 3 (合理) | P/E处于历史中位数区间，PEG 1.5-2.0 |
| 4N (偏贵) | P/E > 历史中位数20-50%，PEG 2.0-3.0 |
| 5N (高估) | P/E > 历史中位数50%+，PEG > 3.0，FCF yield < 无风险利率 |

> 注：成长股估值中枢通常高于传统股，评分应结合行业特性进行判断。

**NVDA各指标对应**：

| 指标 | NVDA当前值 | 评分标准对照 |
|------|-----------|-------------|
| P/E vs 10年中位数（49.73x） | 当前29.84x，低39.9% | ✅ Score 5标准（<中位数20%以上） |
| P/E vs 5年中位数（~55x） | 当前29.84x，低45.7% | ✅ Score 5标准（<中位数20%以上） |
| PEG（3年CAGR） | 0.48 | ✅ Score 5标准（< 1.0） |
| PEG（5年隐含） | 0.60 | ✅ Score 5标准（< 1.0） |
| FCF Yield vs 10Y国债 | 2.05% vs 4.56% | ⚠️ Score 5N标准（FCF yield < 无风险利率） |

**评分判断**：**4分（合理偏低）**

**理由**：
1. **P/E显著低于历史中位数**（-40%至-50%）——这是最有力的低估证据，但部分原因是EPS因AI需求爆发而急剧增长（分母效应），而非股价下跌（分子效应）
2. **PEG远低于1.0**（0.43-0.60）——说明高速增长消化了当前估值，这是成长股估值中最重要的指标
3. **FCF Yield低于无风险利率**——这是传统价值视角的"高估"信号，但需结合成长股特性来看：如果公司保持高增长，未来FCF将快速增长，当前较低的FCF Yield并不意味着永久性的
4. **Forward P/E × FY2027 EPS恰好等于当前股价**——说明市场正在按照"合理价值"定价，既没有显著低估也没有高估
5. **跨期视角**：若以FY2028 EPS和当前Forward P/E计算，隐含股价可达$280（+42%），说明时间站在多头一边
6. 综合评级体系中"成长性"维度权重（25%）大于"估值"维度权重（20%），意味着对成长股的估值应给予更多弹性

> **结论**：NVDA处于 **"合理的成长股估值水平，从历史和PEG角度看偏低"**。Forward P/E 22x对于一个EPS CAGR 46%的公司是合理的。评分给**4分**而非5分，主要考虑到：① DCF绝对估值低于当前股价；② FCF Yield低于无风险利率；③ 超高速增长带来的P/E压缩可能不能永久维持。
