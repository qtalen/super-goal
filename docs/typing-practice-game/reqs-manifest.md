## 12 指版键盘说明

9 个键（代表 10 个手指），底部从左到右排列：

| 键位 | 手指 | 字母列映射 |
|------|------|-----------|
| A | 左小指 | Q, A, Z |
| S | 左无名指 | W, S, X |
| D | 左中指 | E, D, C |
| F | 左食指 | R, F, V, T, G, B |
| J | 右食指 | Y, H, N, U, J, M |
| K | 右中指 | I, K |
| L | 右无名指 | O, L |
| ; | 右小指 | P |
| Space | 双拇指 | Space |
